Overview

Welcome to Project-X! This application allows you to monitor the uptime, page speed, and performance of your websites in real-time.

  • Monitor multiple websites simultaneously
  • Receive alerts and notifications when a website goes down
  • Analyze historical performance data and trends
  • Integrate with other tools and services

Installation

To install Project-X locally on Docker, follow these steps:

  1. Ensure you have Docker Desktop installed on your machine. If not, download and install it from here.
  2. Download the Docker Compose file:
    3. wget https://raw.githubusercontent.com/your-username/website-monitoring-app/main/docker-compose.yml
  3. Run the Docker Compose script:
    4. docker-compose up -d

Once the containers are up and running, you can access the application in your web browser.

Features

Real-time Monitoring

Monitor your websites in real-time to ensure uptime and performance.
